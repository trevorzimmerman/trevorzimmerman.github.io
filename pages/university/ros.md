# ROS Projects

This section showcases robotics projects using the Robot Operating System (ROS and ROS2), including coursework from MTRE 6300 and self-paced projects through The Construct. Projects cover simulation, real robot control, and integration with sensors and manipulators.

This course introduced core ROS concepts, including nodes, topics, tf transformations, SLAM, and motion planning. The video below compiles assignments demonstrating both simulation and real-world applications, including: Turtlesim and TF visualization, ROSbot navigation and mapping in Gazebo, custom two-wheeled robot motion via Python scripts, and FRANKA Panda arm trajectory planning and execution using MoveIt!.

## The Construct – Self-Paced ROS/ROS2 Projects

These projects demonstrate hands-on experience with ROS and ROS2 in simulation and on real robots, including TurtleBot navigation and LIMO-Robot manipulation. Videos highlight pick-and-place operations, autonomous navigation, and object-tracking capabilities.

### TurtleBot ROS Projects

These projects were completed through **The Construct’s online courses**:  
- **ROS Basics in 5 Days (Python)**  
- **ROS2 Basics in 5 Days (Python)**  

Both projects focused on implementing fundamental ROS concepts with a TurtleBot, first in ROS1 (real robot and simulation) and later in ROS2 (simulation only).  

Key components included:  
- Following walls using laser scan data (**topics**)  
- Aligning the TurtleBot with a nearby wall (**services**)  
- Recording odometry during navigation (**actions**)

### LIMO Robot – Mobile Manipulation (ROS2)

Completed through **The Construct’s Mastering Mobile Manipulation with LIMO-Robot** course.  
This course explored combining navigation and manipulation using the LIMO platform, covering MoveIt2 motion planning, object detection, and pick-and-place tasks.  

**LIMO Pick and Place** – The robot detects an object, grasps it, and places it at a new location.
